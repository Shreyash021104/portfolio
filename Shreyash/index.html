cognee.ai

Open-source memory infrastructure for AI agents

Cognee solves the fundamental problem of AI agent amnesia. It ingests unstructured data — documents, conversations, API responses — extracts entities and relationships, and builds a dynamic knowledge graph that agents can query across sessions. Instead of cramming raw context into every prompt, agents retrieve only the precise facts they need.

What makes it compelling for agentic systems: it combines a vector index for semantic similarity with a graph store for relational reasoning — so an agent can ask "what did we decide about auth last week?" and get a structurally accurate answer, not a hallucinated summary.
